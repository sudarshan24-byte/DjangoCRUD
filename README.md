## Installation

Create Virtual Environment

```bash
  pip install virtualenvwrapper-win
  mkvirtualenv example
```

Active Virtual Environment

```bash
  workon example
```

Run Django Project

```bash
  pip install django
  django-admin startproject example
  python manage.py runserver
```

Install django app

```bash
  django-admin startapp example 
```

Run Server

```bash
  python manage.py runserver
```
